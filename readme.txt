== Description ==

Display view count of posts.

Statistic Types;

* Least Viewed for Total
* Least Viewed for Today
* Most Viewed for Today
* Most Viewed for Today
* Last Viewed Posts

Supported cache functions;

* Memcache
* xCache
* Alternative PHP Cache (APC)
* eAccelerator

Support JSON format for ajax applications.

And more :)

== Installation ==

It's too easy :)

1. Upload `sayfa_sayac` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
